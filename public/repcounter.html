Welcome to RepCounter! In order to record you workout progress, start by entering the name of the exercise then click on the Add button. Afterwards, use the slider next to the exercise name to specify how much weight you are currently working with. The minimum and maximum weight values can be adjusted in the options menu. Enter how many reps you perform in the Set area and then click the Submit button. If you wish to edit a set, just click on it. enter the correct value and resubmit.

This web app was created using JavaScript with jQuery, HTML and CSS on the front end. The back end consists of Node.js, Express and SQLite
