Senior Group Project: SpeakUp

A year-long assignment which consisted of proposing an idea for a software program and then planning, designing, documenting and producing it.

  • Cross-platform mobile and web app that could be used by students and teachers to chat during lectures.
  • Gathered requirements, identified use cases, designed architecture and produced requirements analysis document
  • Utilized HTML5, CSS3 and TypeScript with Ionic framework
  • Agile methodology with daily Scrum meetings
  • Final product can be found at SpeakUp.host

Distributed Systems Term Project

The goal of this assignment was to create a program that could synchronize files between 2 or more computers on the same network using peer-to-peer architecture.

  • Coded in Python
  • Synchs files from a designated folder based on which file was most recently modified
  • Multi-threaded: one thread acts as client and another thread acts as server
