Distributed Systems Term Project

The goal of this assignment was to create a program that could synchronize files between 2 or more computers on the same network using peer-to-peer architecture.

  • Coded in Python
  • Synchs files from a designated folger based on which file was most recently modified
  • Multi-threaded: one thread acts as client and another thread acts as server
